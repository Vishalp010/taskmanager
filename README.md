------task manager------
Project Name: Task Manager App
Overview:
A simple task management app where users can register, log in, create tasks, update their status (e.g., "In Progress," "Completed"), and delete tasks

-> Nextjs
-> axios
-> zod 
-> RTK
-> bcryptjs
-> jwt
-> mongoose
-> react-hot-toast
